Online Library Backend (Node.js + Express + MongoDB)
====================================================

**Overview**
------------

This task is a **backend server** for an online library system. It provides **RESTful APIs** for managing books, including adding, updating, deleting, and retrieving book details. The backend is built using **Node.js, Express, and MongoDB** (with Mongoose).

**Features**
------------

The backend supports the following operations:

The backend supports the following operations:

*   **Display all books** – GET /allbooksRetrieves a list of all books from the database.
    

*   **Get a single book** – GET /getbook/:idRetrieves a specific book by its \_id.
    

*   **Add a new book** – POST /addbooksAdds a new book to the database.
    

*   **Update a book** – POST /updatebook/:idUpdates an existing book by its \_id.
    

*   **Delete a book** – POST /deleteBook/:idDeletes a book by its \_id.
    

**Book Data Structure**
-----------------------

Each book document contains the following fields:

*   booktitle (String, required) – The title of the book
    
*   PubYear (Number) – Publication year
    
*   author (String) – Author name
    
*   Topic (String) – Subject/topic of the book
    
*   formate (String) – Either "Electronic" or "Hard Copy"
    
*   \_id (ObjectId) – Automatically generated by MongoDB
